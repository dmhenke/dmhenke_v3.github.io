// who I am

Biologist who codes.
Statistician who reads papers.

I am a computational biologist at Baylor College of Medicine, where my work sits at the intersection of statistical machine learning, network biology, and functional genomics.

High-dimensional biological data is noisy, collinear, and humbling. Standard approaches will happily select a feature simply because it lives next to the real driver. My central research question is whether we can teach models what biology already knows — and use that knowledge to find signals that statistics alone would miss.

Practically, this means building regularized regression frameworks that incorporate protein–protein interaction networks, applying them to large-scale cancer dependency screens from DepMap, and translating the results toward therapeutic hypotheses in precision oncology.

My primary tool is R. My primary motivation is the gap between a statistically significant finding and a biologically meaningful one.

// featured project

BioPrimeLASSO

BioPrimeLASSO
Bio-Primed LASSO · Biomarker Discovery · R Package
📄 DOI 10.1038/s41698-025-00825-9 ★ GitHub GPL-3.0

Standard LASSO is statistically elegant but biologically blind. In collinear omics data — such as copy number profiles subject to linkage disequilibrium — it will select a gene simply because it lives next door to the real driver.

BioPrimeLASSO solves this by incorporating protein–protein interaction (PPI) network scores from STRING DB directly into the LASSO regularization penalty. A tunable parameter Φ, optimized by cross-validation, controls how strongly biological prior knowledge steers feature selection.

Applied to DepMap dependency data, the method recovered GAB2 as an EGFR dependency biomarker — a finding missed by standard approaches and the DepMap Predictability portal — and linked it to clinical sensitivity to EGFR inhibitors.

Installation
# requires devtools
install.packages("devtools")
devtools::install_github(
  "dmhenke/BioPrimeLASSO"
)
Core usage
# Compute PPI scores for gene of interest
scores  <- get_scores(
  gene    = "EGFR",
  network = ppi
)

# Run bio-primed LASSO
results <- bplasso(
  scale(X), y, scores,
  n_folds   = 10,
  phi_range = seq(0, 1, length = 30)
)

What I think about

⚖️
Regularized Regression

Biologically-informed LASSO, adaptive penalties, and penalty weighting with network priors. Extending classical shrinkage estimators with domain knowledge.

🧫
Functional Genomics

DepMap CRISPR and RNAi dependency screens, DEMETER2, and Chronos. Mapping which genes cancer cells cannot live without.

🕸️
Network Biology

PPI network integration via STRING DB, co-dependency analysis, and synthetic lethality. Treating interaction graphs as prior knowledge, not decoration.

🎯
Precision Oncology

Biomarker discovery from copy number variation, gene dependency, and drug sensitivity data. Bridging the gap between statistical rigor and clinical relevance.

📐
High-Dimensional Statistics

Feature selection in p ≫ n regimes, collinearity in omics data, and reproducible analysis pipelines. Making statistics behave when the data does not.

🔬
Translational Bioinformatics

Turning computational predictions into testable biological hypotheses. Finding biomarkers that are not just significant, but meaningful.
